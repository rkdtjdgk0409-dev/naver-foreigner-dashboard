노션용 소형 화면 적용 방법

1. GitHub 저장소의 Code 화면으로 이동합니다.
2. 기존 index.html을 클릭합니다.
3. 오른쪽 위 휴지통 아이콘으로 삭제하지 말고,
   Add file → Upload files를 누릅니다.
4. 이 폴더 안의 index.html을 업로드합니다.
5. 같은 이름의 파일을 교체한다는 안내가 나오면 그대로 진행합니다.
6. Commit changes를 누릅니다.
7. 약 1~3분 후 GitHub Pages 주소를 새로고침합니다.
8. 노션에서도 임베드 블록을 새로고침합니다.
9. 노션 페이지 오른쪽 위의 ··· → 전체 너비를 켭니다.
10. 임베드 블록 높이를 약 430~470px 정도로 맞춥니다.

변경 내용
- 종목명 앞의 '관심종목 추가(숫자)' 문구 자동 제거
- 순매수/순매도 내용 유지
- 차트와 표를 좌우로 배치
- 제목, 여백, 글씨, 표 높이 축소
